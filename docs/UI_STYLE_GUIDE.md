# UI Style Guide

> **Project:** 小书童 (AI Family Teacher)  
> **Last updated:** 2026-04-13  
> **Design System:** iOS-inspired Chat-Centric UI

---

## Overview

小书童采用 **iOS 风格的设计系统**，以聊天为中心（Chat-Centric），面向儿童用户。整体风格干净、现代、温暖，避免冷冰冰的商务感。

---

## 设计原则

### 1. 白色主基调
- 纯白背景 `#FFFFFF`
- 避免渐变（仅在 Landing 页使用）
- 干净、清爽、符合 iOS 原生应用风格

### 2. 温暖友好
- 面向儿童用户，避免冷冰冰的文案
- 空状态使用 Emoji + 温暖问候
- 快捷按钮使用温暖色调 + Emoji 图标

### 3. 最大化内容空间
- 移除底部导航栏
- 移除顶部 Component Switcher Bar
- 通过汉堡菜单访问次要功能
- 聊天区占据屏幕主要空间

### 4. 毛玻璃效果
- Header 和输入区使用 iOS 风格毛玻璃（BackdropFilter）
- 滚动时内容隐约可见
- 透明度 85%-90%

## 阴影系统

### 消息气泡阴影（对角线浅色阴影）
```dart
boxShadow: [
  BoxShadow(
    color: Color(0x08000000),  // 极淡黑色
    offset: Offset(2, -2),      // 向右上偏移
    blurRadius: 6,
    spreadRadius: 0,
  ),
]
```

### 卡片/按钮微妙阴影
```dart
boxShadow: [
  BoxShadow(
    color: Color(0x0A000000),
    offset: Offset(0, 1),
    blurRadius: 3,
    spreadRadius: 0,
  ),
]
```

### Logo 发光阴影
```dart
boxShadow: [
  BoxShadow(
    color: Color(0xFFFF6B8B).withOpacity(0.3),
    blurRadius: 16,
    offset: Offset(0, 6),
  ),
]
```

| 级别 | 大小 | 字重 | 用途 |
|------|------|------|------|
| **大标题** | `24px` | 700 | 空状态问候语 |
| **标题 1** | `17px` | 600 | Header 标题 |
| **正文** | `16px` | 400 | 一般文本 |
| **副标题** | `15px` | 500 | 快捷按钮文本 |
| **脚注** | `13px` | 400 | 提示文字 |

### 字体特性
- 字间距：`-0.4`（紧凑感，iOS 风格）
- 行高：`1.4`（适中，易读）
- 颜色：`#000000`（主文本）、`#3C3C43`（次要）、`#8E8E93`（三级）

## 组件样式

### 1. Header（顶部导航）

```
┌─────────────────────────────────────────────────────────┐
│  ☰    与小书童对话                    📜    👤       │
│  [无背景框，毛玻璃效果]                                  │
└─────────────────────────────────────────────────────────┘
```

**特性：**
- 毛玻璃背景（`blur: 10.0, opacity: 0.85`）
- 底部细线分隔（`0.5px, #E5E5EA`）
- 所有图标作为纯 Icon（无 IconButton 外框）
- `crossAxisAlignment: center` 确保垂直对齐
- 左侧汉堡菜单：`Icons.menu_rounded, size: 26`
- 右侧图标：历史 `size: 24`、加号 `size: 26`、头像 `size: 32`

### 2. 消息气泡

**AI 气泡（左侧）：**
- 背景色：`#F2F2F7`（浅灰）
- 圆角：`20px`（上）/ `4px`（左下）/ `20px`（右下）
- 阴影：对角线浅色阴影 `(2, -2)`
- 内边距：`14px × 10px`
- 最大宽度：`75%` 屏幕宽度

**用户气泡（右侧）：**
- 背景色：`#007AFF`（iOS 蓝）
- 圆角：`20px`（上）/ `4px`（右下）/ `20px`（左下）
- 阴影：对角线浅色阴影 `(2, -2)`
- 文字颜色：白色
- 头像：右侧显示

### 3. 输入区

```
┌─────────────────────────────────────────────────────────┐
│  📷 🎤 🔊 [说说你的想法...              ] [↑]          │
│  [毛玻璃背景，所有按钮同一行]                            │
└─────────────────────────────────────────────────────────┘
```

**特性：**
- 毛玻璃背景（`blur: 10.0, opacity: 0.9`）
- 顶部细线分隔（`0.5px, #E5E5EA`）
- 左侧按钮：图片 `📷`、麦克风 `🎤`、音量 `🔊`（方形，32×32）
- 输入框：圆角矩形 `20px`，浅灰背景 `#F2F2F7`
- 发送按钮：蓝色圆形 `36px`，↑ 箭头图标
- Placeholder：`说说你的想法...`

### 4. 空状态

```
         ✨
    [温暖渐变圆形 Logo]

   👋 嗨，小明！
   今天想学点什么新东西呀？

   💡 帮我解释一个概念  ← 暖橙背景
   ✏️ 出一些练习题    ← 浅绿背景
   🌟 给我学习小建议   ← 淡紫背景
```

**Logo：**
- 渐变背景：`#FF9A56 → #FF6B8B`
- 图标：`Icons.auto_awesome_rounded, size: 48, white`
- 阴影：粉色发光 `(0, 6) blur: 16`
- 尺寸：`88×88`

**文案：**
- 问候语：`👋 嗨，${studentName}！`（`22px, 700`）
- 副标题：`今天想学点什么新东西呀？`（`15px, 500`）

**快捷按钮：**
- 无外框，仅背景色
- Emoji 图标 + 文字
- 大圆角 `20px`
- 三种温暖背景色区分

### 5. 汉堡菜单（底部抽屉）

**触发方式：** 点击 Header 左侧 ☰ 图标

**样式：**
- 白色背景，圆角顶部 `24px`
- 顶部指示条（`40×4px, #E5E5EA`）
- 菜单项：图标 + 文字 + 激活态 ✓ 标记
- 分隔线：`#E5E5EA`
- 当前页面高亮：蓝色背景 + 蓝色文字

### 6. 快捷提示按钮（Prompt Cards）

**样式：**
- 无外框，仅背景色
- 圆角 `20px`
- Emoji 图标（`20px`）+ 文字
- 三种背景色区分
- 内边距：`18px × 14px`

## 动画与交互

### 滚动
- 弹性滚动：`BouncingScrollPhysics()`（iOS 风格）
- ListView 自动滚动到底部

### 毛玻璃
- `BackdropFilter` + `ImageFilter.blur(sigmaX: 10, sigmaY: 10)`
- Header 和输入区实时模糊背景

### 按钮反馈
- `InkWell` 涟漪效果
- 圆角 `BorderRadius.circular(radiusLarge)`

## 图标规范

### 图标风格
- 使用 `_rounded` 后缀的 Material Icons（iOS 风格圆角）
- 例：`Icons.menu_rounded`, `Icons.add_rounded`, `Icons.history_rounded`

### 图标尺寸
| 位置 | 尺寸 |
|------|------|
| Header 菜单图标 | `24-26px` |
| 输入区按钮图标 | `18px` |
| 发送按钮图标 | `18px` |
| 快捷按钮 Emoji | `20px` |
| 空状态 Logo 图标 | `48px` |
| 头像 | `32-36px` |

## 文案风格

### 语气
- **温暖亲切**，面向儿童
- 使用 Emoji 增加趣味
- 避免冷冰冰的商务用语

### 示例

| 场景 | ❌ 避免 | ✅ 推荐 |
|------|---------|---------|
| 空状态 | "开始对话" | "👋 嗨，小明！今天想学点什么新东西呀？" |
| 输入框 | "输入问题..." | "说说你的想法..." |
| 快捷按钮 | "帮我解释这个概念" | "💡 帮我解释一个概念" |
| 加载中 | "处理中" | "小书童正在思考..." |
