# 三个组件设计：黑板、作业本、笔记本

## 组件概览
- **黑板**：AI讲解展示，孩子手写备注
- **作业本**：AI出题，孩子答题，AI批改  
- **笔记本**：孩子记笔记，AI自动整理
- **显示模式**：分时显示，一次只显示一个，支持手动切换

---

## 1. 黑板设计

### 功能
- AI通过function calling写入教学内容（解题步骤、图形、标注）
- 孩子手写备注、画圈提问
- 支持多页（翻页看步骤1、步骤2...）

### 视觉属性
- 背景：深灰#2E2E2E或黑板绿#0A5C36
- 笔迹：粉笔效果，多色可选（白、黄、红、绿）
- 辅助线：无或浅色网格
- 工具：粉笔、板擦、翻页箭头

### 内容结构
```json
{
  "type": "blackboard",
  "pages": [
    {
      "title": "分数除法步骤1",
      "elements": [
        {"type": "text", "content": "除以分数 = 乘以倒数", ...},
        {"type": "arrow", "from": "...", "to": "...", "label": "取倒数"}
      ]
    }
  ],
  "student_notes": []  // 孩子手写笔迹
}
```

## 2. 作业本设计

### 功能
- AI出题：显示题目、答题区、提示
- 孩子答题：手写答案、计算过程
- AI批改：显示对错、批注、正确答案
- 支持多题：上一题/下一题导航

### 视觉属性  
- 背景：米黄#F8F5E6，带浅灰色横线（间距24px）
- 笔迹：铅笔/钢笔效果，黑色/蓝色
- 答题区：虚线框标注
- 批改痕迹：红色笔迹（AI批注）、绿色对勾/红叉

### 内容结构
```json
{
  "type": "workbook",
  "current_question": 1,
  "total_questions": 8,
  "question": {
    "text": "2/3 ÷ 1/4 = ?",
    "answer_area": {"x": 50, "y": 100, "width": 200, "height": 80}
  },
  "student_answer": "8/3",  // 识别的文字或笔迹数据
  "ai_feedback": {
    "correct": true,
    "score": 1.0,
    "comment": "正确，可写成带分数2又2/3",
    "correction_marks": []  // 批改笔迹数据
  }
}
```

## 3. 笔记本设计

### 功能
- 孩子自由手写/打字记笔记
- 按知识点自动分类（数学、语文、错题笔记...）
- AI自动整理：提取关键词、生成摘要、关联相关题目
- 支持搜索和回顾

### 视觉属性
- 背景：白色#FFFFFF，无格线或极浅点阵
- 笔迹：钢笔效果，多种颜色
- 页面标签：顶部显示知识点标签
- 工具：笔、荧光笔、橡皮、文本框、标签按钮

### 内容结构
```json
{
  "type": "notebook",
  "page_id": "2024-01-15-math",
  "tags": ["分数除法", "易错点"],
  "content": {
    "strokes": [],  // 手写笔迹
    "text_blocks": [
      {"text": "分数除法要取倒数", "type": "要点"},
      {"text": "注意符号变化", "type": "提醒"}
    ]
  },
  "ai_summary": {
    "keywords": ["取倒数", "符号", "带分数"],
    "related_questions": ["q123", "q456"],
    "suggested_review_date": "2024-01-22"
  }
}
```

## 切换逻辑

### 默认自动切换规则
1. AI开始讲解 → 切换到黑板
2. AI出题/孩子要答题 → 切换到作业本  
3. 孩子说"记一下"或点击笔记按钮 → 切换到笔记本
4. 当前任务完成 → 根据下一动作决定组件

### 手动切换控件
```
顶部控制栏（始终显示）：
当前：[🎯 黑板] [📝 作业本] [📓 笔记本]

或底部快捷栏：
[黑板] [作业本] [笔记本] [对话]
```

### 状态保持
- 切换时保存当前组件状态（内容、笔迹）
- 可快速返回之前组件继续
- 历史记录：可查看过去使用记录

## 技术实现要点

### 1. 基础绘制引擎
- 共用`DrawingSurface`基类：笔迹捕获、渲染、序列化
- 矢量笔迹存储：记录坐标、压力、时间戳
- 实时渲染：60fps平滑绘制

### 2. 组件管理器
- 管理三个组件实例
- 处理切换动画（淡入淡出或滑动）
- 状态序列化/反序列化

### 3. AI集成
- **黑板**：AI通过`update_blackboard` function call写入内容
- **作业本**：AI通过`set_question`出题，通过`check_answer`批改
- **笔记本**：AI通过`analyze_notes`分析笔记，生成整理结果

### 4. 内容识别
- 手写文字识别（OCR）：将笔迹转为文字供AI处理
- 简单图形识别：直线、圆形、箭头
- 笔迹分类：区分孩子笔迹和AI笔迹

### 5. 数据同步
- 实时保存到本地数据库
- 笔迹增量保存，避免数据丢失
- 支持离线使用，联网时同步

### 6. 性能优化
- 画布分层：背景、AI内容、孩子笔迹分不同层
- 笔迹压缩：存储时压缩坐标数据
- 按需渲染：不可见组件暂停渲染
