# AI家庭教师 - 核心设计整合

## 四个核心Workflow

### 1. 回答问题（答题）
**场景**：孩子遇到难题求助，AI引导式讲解
**关键交互**：
- 孩子提问（语音/拍照/文字）
- AI询问是否尝试过
- 从类似简单题开始引导
- 使用黑板展示解题步骤
- 记录错题，安排回访

### 2. 按需出题（出题）
**场景**：孩子主动要求练习，AI个性化出题
**关键交互**：
- 孩子说明需求（"分数除法我老错"）
- AI分析历史数据，推荐专项练习
- 作业本显示题目，孩子答题
- 即时批改反馈，错误时黑板讲解
- 统计进步，安排巩固计划

### 3. 上课/补习
**场景**：孩子系统学习某个知识点
**关键交互**：
- AI制定课程计划（概念→规则→技巧→练习）
- 黑板全程展开授课，穿插提问
- 孩子手写参与，实时练习
- 课程结束测试，掌握度评估
- 记录学习路径，建议后续学习

### 4. 整理笔记
**场景**：孩子自由记录学习笔记，AI自动整理
**关键交互**：
- 笔记本组件自由手写/打字
- 按知识点自动分类标签
- AI提取关键词，生成摘要
- 关联相关题目和错题
- 智能提醒复习

## 三个核心组件

### 1. 黑板
- **用途**：AI讲解展示
- **背景**：深灰#2E2E2E或黑板绿#0A5C36
- **笔迹**：粉笔效果，多色
- **控制**：AI通过function calling写入内容
- **交互**：孩子手写备注，画圈提问

### 2. 作业本
- **用途**：孩子答题，AI批改
- **背景**：米黄#F8F5E6，浅灰色横线
- **笔迹**：铅笔/钢笔效果
- **控制**：AI出题，孩子答题提交
- **交互**：答题区，提交按钮，批改痕迹

### 3. 笔记本
- **用途**：孩子记笔记，AI整理
- **背景**：白色#FFFFFF，无格线或点阵
- **笔迹**：钢笔效果，多色
- **控制**：孩子自由记录
- **交互**：标签分类，AI分析整理

## 组件切换规则
- **分时显示**：一次只显示一个组件
- **自动切换**：AI根据场景选择合适组件
- **手动切换**：顶部控制栏 [🎯黑板][📝作业本][📓笔记本]
- **状态保持**：切换时保存内容，可快速返回

## 技术架构

### 前端
- **框架**：Flutter 3.0+（支持iOS、Android、Web、桌面）
- **状态管理**：Provider/Riverpod
- **数据库**：SQLite（sqflite）
- **网络**：http/dio
- **绘图**：CustomPaint + GestureDetector

### AI集成
- **LLM API**：DeepSeek或其他
- **Function Calling**：控制组件内容更新
- **手写识别**：基础形状识别 + 可选OCR
- **数据同步**：本地存储 + 云端备份

### 核心数据结构
```json
// 黑板内容
{"type": "blackboard", "pages": [], "student_notes": []}

// 作业本题目
{"type": "workbook", "question": {}, "student_answer": "", "ai_feedback": {}}

// 笔记本笔记
{"type": "notebook", "tags": [], "content": {}, "ai_summary": {}}
```

## 底层配套功能
1. **用户档案**：年级、教材版本、学习历史
2. **错题本**：自动记录错题，按知识点分类
3. **学习统计**：用时、正确率、掌握度趋势
4. **家长端**：学习报告、抽查测试、目标设置
5. **同步系统**：多设备数据同步
