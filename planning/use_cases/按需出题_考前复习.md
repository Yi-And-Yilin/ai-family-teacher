# 用例：按需出题 - 考前复习

## 场景描述
明天数学要考试，小明想重点复习分数运算单元。AI根据小明的历史错题和单元知识点，生成个性化复习计划。

## 参与角色
- **学生**：小明
- **AI家庭教师**：系统

## 前置条件
1. 小明已使用app学习至少一周
2. 系统记录了小明的错题历史和知识点掌握情况
3. 小明知道明天考试的单元范围

## 基本流程

### 步骤1：启动复习模式
1. 小明打开app，点击"我要复习"按钮
2. 选择"考前复习"选项
3. 输入考试信息："明天考数学第三单元 - 分数运算"

### 步骤2：智能诊断
**系统分析历史数据后显示：**
```
📊 你的分数运算掌握情况：
✅ 已掌握：分数加减法（通分熟练）
⚠️ 需巩固：分数乘除法（容易混淆）
❌ 薄弱点：带分数与假分数转换

📈 错题统计：
- 分数加减法：2次错误（已掌握）
- 分数乘法：5次错误
- 分数除法：7次错误
- 带分数转换：8次错误
```

### 步骤3：生成复习计划
**系统**："根据你的情况，建议按以下顺序复习："
```
🎯 今日复习计划（预计60分钟）：
1. 带分数转换专项练习（20分钟）- 最高优先级
2. 分数除法巩固练习（15分钟）
3. 分数乘法概念梳理（10分钟）
4. 综合测试（15分钟）- 检验复习效果
```

### 步骤4：专项练习 - 带分数转换
**系统**："我们先攻克最薄弱的环节。看这道题：2又3/4 写成假分数是多少？"

**小明**："11/4？"

**系统**："正确！计算过程：2×4+3=11，分母不变是4。再来一题：3又1/2 = ?"

**小明**："7/2。"

**系统**："很好！现在反过来：13/5 写成带分数？"

**小明**："2又3/5。"

**系统**："完全正确！你掌握了基本方法。我们加快难度..."

### 步骤5：分数除法巩固
**系统**："复习分数除法规则：除以分数 = 乘以倒数。2/3 ÷ 1/4 = ?"

**小明**："2/3 × 4/1 = 8/3 = 2又2/3。"

**系统**："正确！注意符号变化：3/4 ÷ (-1/2) = ?"

**小明**："3/4 × (-2/1) = -6/4 = -1又1/2。"

### 步骤6：概念梳理
**系统**："我们梳理一下容易混淆的点："
- "分数乘法：分子乘分子，分母乘分母"
- "分数除法：变除为乘，取倒数"
- "带分数→假分数：整数×分母+分子"
- "假分数→带分数：分子÷分母=商...余数"

### 步骤7：综合测试
**系统**："最后做个15题的小测试，检验复习效果："
1. "1又2/3 × 3/5 = ?"
2. "4/7 ÷ 2/3 = ?"
3. "5/8 = ?/32"
...

**小明完成测试，系统立即批改并讲解错题。**

### 步骤8：复习总结
**系统**："今日复习完成！成绩对比："
```
复习前掌握度：65%
复习后测试：85%
提升：20个百分点

🎯 明日考试建议：
1. 重点检查带分数答案格式
2. 分数除法记得取倒数
3. 预留时间检查计算
```

## 备选流程

### 备选A：时间有限
**小明**："我只有30分钟复习。"

**系统**："调整为高效复习模式："
1. "只做最薄弱点的5道题（10分钟）"
2. "快速过易错点（10分钟）"
3. "5题模拟测试（10分钟）"

### 备选B：已经掌握很好
如果系统分析发现小明掌握度>90%：
**系统**："你在这个单元掌握得很好！建议："
1. "做3道难题挑战自己"
2. "预习下一单元"
3. "或者休息，保持状态"

## 后置条件
1. 完成个性化复习计划
2. 系统更新掌握度评分
3. 生成复习报告
4. 错题进入回访计划（考后一周再测试）

## 业务规则
1. **优先级排序**：错题频率最高的知识点优先复习
2. **时间分配**：根据剩余时间和掌握度动态调整计划
3. **难度递进**：从基础题开始，逐步增加难度
4. **即时反馈**：每道题都有讲解，即时纠正错误
