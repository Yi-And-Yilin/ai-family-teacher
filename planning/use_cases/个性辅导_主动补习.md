# 用例：个性辅导 - 主动补习

## 场景描述
小明感觉分数除法学得不好，主动向AI请求补习。AI分析小明的错题历史，设计个性化补习课程。

## 参与角色
- **学生**：小明（主动发起）
- **AI家庭教师**：系统

## 前置条件
1. 小明最近在分数除法上多次出错
2. 小明有自主补习的意愿
3. 系统有足够的历史数据用于分析

## 基本流程

### 步骤1：发起补习请求
1. 小明打开app，点击"我要补习"按钮
2. 选择补习方向：
   - ○ 薄弱知识点
   - ○ 本周错题
   - ○ 指定内容
3. 小明选择"薄弱知识点"，输入："分数除法"

### 步骤2：智能诊断分析
**系统分析后显示诊断报告：**
```
📋 分数除法诊断报告
📅 分析周期：最近2周
📊 总体掌握度：60%（需加强）

🔍 具体问题分析：
1. 概念混淆：40%错误因"忘记取倒数"
2. 符号处理：30%错误在"负数运算"
3. 结果化简：20%错误在"未约分"
4. 格式错误：10%错误在"带分数格式"

🎯 建议补习重点：
1. 核心概念：除以分数 = 乘以倒数
2. 符号规则：负数除法符号变化
3. 化简习惯：结果必须约分
```

### 步骤3：制定补习计划
**系统**："根据诊断，建议30分钟补习计划："
```
⏰ 补习计划（30分钟）
1. 概念梳理：分数除法规则（5分钟）
2. 基础练习：正数除法（10分钟）
3. 进阶练习：负数除法（10分钟）
4. 综合测试：检验效果（5分钟）
```

**小明**："好的，开始吧。"

### 步骤4：概念梳理
**系统**："我们先明确核心规则：除以一个分数等于乘以它的倒数。"
- "公式：a/b ÷ c/d = a/b × d/c"
- "例子：2/3 ÷ 1/4 = 2/3 × 4/1 = 8/3"

**系统**："为什么这样？用实际例子理解：2/3 ÷ 1/4 意思是，2/3里面有多少个1/4。"

**系统**："画图解释：2/3 = 8/12，1/4 = 3/12，8/12 ÷ 3/12 = 8÷3 = 8/3。"

### 步骤5：基础练习
**系统**："现在做几道基础题巩固："
1. "3/4 ÷ 1/2 = ?"
2. "5/6 ÷ 2/3 = ?"
3. "7/8 ÷ 1/4 = ?"

**小明依次作答，系统即时反馈：**
- 第一题正确：3/4 × 2/1 = 6/4 = 3/2
- 第二题正确：5/6 × 3/2 = 15/12 = 5/4
- 第三题错误：7/8 × 4/1 = 28/8 = 7/2（小明忘了化简）

**系统**："第三题计算正确，但28/8要化简为7/2或3又1/2。养成约分习惯。"

### 步骤6：进阶练习（负数）
**系统**："现在加入负数，注意符号规则："
1. "2/3 ÷ (-1/4) = ?"
2. "(-3/4) ÷ 1/2 = ?"
3. "(-5/6) ÷ (-2/3) = ?"

**系统讲解符号规则：**
- "正÷负 = 负"
- "负÷正 = 负"
- "负÷负 = 正"

### 步骤7：综合测试
**系统**："最后5题测试，检验补习效果："
1. "3/5 ÷ 2/3 = ?"
2. "(-4/7) ÷ 1/2 = ?"
3. "1又1/2 ÷ 3/4 = ?"（带分数）
4. "2/3 ÷ 0.25 = ?"（小数分数混合）
5. "应用题：小明有3/4升果汁，每杯装1/6升，可分几杯？"

### 步骤8：补习总结
**系统**："补习完成！成绩分析："
```
📊 补习前后对比：
- 基础题正确率：60% → 100%
- 符号题正确率：40% → 80%
- 综合题正确率：50% → 70%

🎯 掌握度提升：60% → 85%
⏰ 下次建议：3天后复习一次
```

**系统**："主要进步："
1. "明确了'除以分数=乘以倒数'的核心规则"
2. "掌握了负数除法的符号处理"
3. "仍需加强：带分数除法和应用题理解"

## 备选流程

### 备选A：补习中途放弃
**小明**："我累了，想休息。"

**系统**："好的，保存当前进度。已学习15分钟，完成基础部分。"
- "保存进度：概念梳理+基础练习完成"
- "下次可从'负数除法'继续"
- "建议24小时内完成剩余部分"

### 备选B：需要更多练习
**小明**："我还是不太明白负数除法。"

**系统**："我们多做一些符号题，用具体数字理解："
- "6 ÷ (-2) = -3 （正÷负=负）"
- "(-6) ÷ 2 = -3 （负÷正=负）"
- "(-6) ÷ (-2) = 3 （负÷负=正）"

**系统**："分数同理：符号规则不变。"

1. **诊断优先**：基于数据诊断，不是随机练习
2. **难度递进**：从概念到基础到综合
3. **即时反馈**：每道题都有讲解
4. **进度保存**：支持中断续学
5. **效果评估**：补习前后对比量化
