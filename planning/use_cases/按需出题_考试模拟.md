# 用例：按需出题 - 考试模拟

## 场景描述
孩子想要检验某个章节或知识点的掌握情况，要求AI生成模拟测试题，进行综合性练习。

## 参与角色
- **学生**：小明（小学五年级）
- **AI家庭教师**：系统

## 前置条件
1. 孩子明确要测试的知识点范围
2. 系统了解孩子的年级和教材版本

## 核心流程

### 1. 测试范围指定
**孩子指定测试范围：**
- **语音指定**："出20道第三单元的题"、"做个分数运算模拟考"
- **选择指定**：从知识点列表中选择章节或主题
- **组合指定**："覆盖分数加减乘除，重点出除法题"

**系统确认范围：**
- "好的，生成分数运算第三单元测试，包含加减乘除，共20题。"
- "需要设置时间限制吗？（如30分钟）"
- "难度分布：基础题40%，中等题40%，难题20%"

### 2. 试卷生成
**AI智能组卷：**
1. **全面覆盖**：确保测试范围所有知识点都有题目
2. **难度梯度**：基础题、中等题、难题合理分布
3. **题型多样**：计算题、概念题、应用题、判断题等
4. **避免重复**：与历史错题和练习题目区分
5. **时间估算**：根据题量和难度估算合理用时

**试卷预览：**
- 显示题目数量、预估时间、知识点分布
- 孩子可调整："减少5道题"、"增加应用题比例"
- 确认后开始模拟考试

### 3. 模拟考试环境
**考试模式设置：**
- **计时模式**：可选时间限制（如30分钟倒计时）
- **全屏模式**：减少干扰，专注答题
- **草稿纸功能**：提供虚拟草稿纸或拍照上传计算过程
- **进度显示**：显示已完成题目和剩余时间

**答题交互：**
- 语音读题（可选）
- 文字题目展示
- 多种答题方式：语音回答、文字输入、拍照上传答案
- 支持"跳过"功能，稍后返回

### 4. 提交与批改
**提交试卷：**
- 时间到自动提交，或手动提前提交
- 系统立即批改，显示成绩

**成绩分析：**
```
📊 模拟测试结果
✅ 正确：16/20题
📈 得分：80分
⏱️ 用时：28分钟（预估30分钟）
🎯 知识点掌握情况：
  - 分数加减法：100% ✅
  - 分数乘法：75% ⚠️  
  - 分数除法：60% ❌
  - 应用题：85% ✅
```

### 5. 错题讲解
**逐题讲解：**
- 按错误率排序讲解错题
- 每道错题分析错误原因
- 提供正确解法和类似题巩固
- 支持语音讲解和文字解析

**综合建议：**
- "分数除法是主要薄弱点，建议专项练习"
- "应用题表现良好，保持优势"
- "时间分配合理，没有题目来不及做"

## 交互示例

### 场景：分数运算单元测试
**小明**："出20道分数运算的题，我想模拟考试。"

**系统**："好的，生成分数运算综合测试，覆盖加减乘除和应用题，预估时间30分钟。开始吗？"

**小明**："开始。"

（30分钟模拟考试，小明完成答题）

**系统批改后：**
"测试完成！得分85分，20题对了17题。主要错在分数除法（3题错2题）。"

**错题讲解：**
1. "第8题：2/3 ÷ 1/4 = 8/3，你写成了2/3 × 1/4 = 2/12。记住：除变乘，取倒数。"
2. "第15题：应用题单位换算错误，注意升和毫升的转换。"

**后续建议：**
"建议：1) 分数除法专项练习5道题 2) 单位换算概念复习 3) 3天后重测类似题目"

## 关键业务规则

1. **范围明确**：测试范围必须清晰指定
2. **全面覆盖**：确保测试范围所有知识点都有体现
3. **难度合理**：基础、中等、难题比例适当
4. **模拟真实**：提供考试环境和时间压力
5. **深度分析**：不仅批改对错，还要分析错误原因和薄弱点

## 测试类型变体

### 1. 章节测试
- "出第三单元测试题"
- 覆盖单个章节所有知识点
- 题量适中（15-25题）

### 2. 知识点专项测试  
- "分数除法出10道题"
- 聚焦单个知识点深度测试
- 难度递进，从基础到综合

### 3. 综合模拟考
- "期中考试模拟"
- 多章节综合，题量较大（30-50题）
- 严格计时，模拟真实考试

### 4. 快速抽查
- "5分钟快测，出5道题"
- 随机抽查，快速检验
- 适合碎片时间练习

## 数据记录
- 测试范围、题量、用时
- 各知识点正确率和掌握度
- 错误类型分布
- 时间分配分析（哪些题用时过长）
- 历史测试成绩趋势

## 家长端同步
- 推送测试完成通知和成绩
- 查看详细测试报告和错题分析
- 可查看孩子的时间管理情况
- 发起特定范围的抽查测试
- 设置定期模拟考试计划
