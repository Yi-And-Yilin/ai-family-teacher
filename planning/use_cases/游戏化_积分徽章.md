# 用例：游戏化激励 - 积分徽章系统

## 场景描述
小明完成今日学习后，系统自动计算积分奖励，解锁新徽章，激发学习动力。

## 参与角色
- **学生**：小明
- **系统**：AI家庭教师

## 前置条件
1. 小明已完成今日学习任务
2. 系统已记录学习数据（正确率、时长、主动性等）
3. 小明已有一定的积分和徽章基础

## 基本流程

### 步骤1：学习结束，积分结算
小明完成45分钟学习后，系统自动弹出：

**🎉 今日学习完成！积分结算：**
```
⏰ 学习时长：45分钟 → +45积分
✅ 作业正确率：85% → +25积分（高于80%奖励）
📚 复习完成：15题 → +15积分
💪 主动提问：3次 → +15积分（鼓励主动性）
🎯 错题掌握：回访正确 → +20积分

💰 今日总计：+120积分
📈 累计积分：2,850 → 2,970
```

### 步骤2：成就解锁
**系统**："恭喜达成新成就！"
```
🏆 新解锁徽章：
1. 【连续学习之星】 - 连续7天完成学习任务
   "持之以恒，未来可期！"
   
2. 【错题终结者】 - 单日错题回访正确率100%
   "曾经的难点，今日的强项！"

📊 徽章墙更新：
- 已有徽章：8枚
- 总成就值：350点
- 排名：班级前20%
```

### 步骤3：进度可视化
**系统展示学习进度：**
```
📈 学习成长树：
🌱 基础技能：分数运算 ⭐⭐⭐⭐☆（4.2/5）
🌿 进阶技能：应用题理解 ⭐⭐☆☆☆（2.5/5）
🌳 综合能力：数学思维 ⭐⭐⭐☆☆（3.0/5）

🎯 距离下一等级：还需30积分
🔥 连续打卡：第7天（再坚持3天有额外奖励）
```

### 步骤4：奖励兑换
**系统**："积分可兑换："
```
🛒 奖励商店：
1. 【虚拟装扮】学习伙伴新皮肤 - 300积分
2. 【学习特权】优先体验新功能 - 500积分
3. 【实体奖励】数学故事书（需家长确认） - 1,000积分
4. 【能力解锁】自定义学习主题 - 800积分

💡 当前推荐：学习伙伴新皮肤（300积分）
   "让你的AI老师换上新装！"
```

**小明选择兑换新皮肤**：
- 确认扣除300积分
- 新皮肤立即生效
- AI老师外观变化，增加新鲜感

### 步骤5：社交激励（匿名）
**系统展示激励信息：**
```
👥 学习社区动态（匿名处理）：
- "同学A刚刚解锁了'速度之王'徽章"
- "本周进步最快：同学B（正确率提升25%）"
- "热门挑战：分数运算速算赛（已有50人参与）"

🎮 可选参与：
1. 每日挑战：10分钟速算赛
2. 周度排名：同年级积分榜（仅显示前20名）
3. 成就对比：与相似水平同学比较
```

### 步骤6：个性化鼓励
**根据小明今日表现，AI生成鼓励语：**
```
🌟 今日亮点：
1. 主动求助3次 → "不懂就问，是学霸品质！"
2. 错题回访全对 → "温故知新，真正掌握了！"
3. 坚持45分钟 → "专注力超越90%同龄人！"

🎯 明日目标：
"再坚持2天，解锁'十日战士'徽章！
重点攻克：应用题理解（当前2.5星→目标3星）"
```

## 备选流程

### 备选A：学习时长不足
如果小明只学了15分钟：
**系统**："今日学习时长较短，继续加油！"
```
⏰ 学习时长：15分钟 → +15积分
📊 建议：明日目标30分钟
💡 鼓励："千里之行始于足下，明天继续！"

🎯 连续打卡提醒：第6天（中断将重置）
```

### 备选B：正确率较低
如果正确率<60%：
**系统**："今日遇到了一些困难，没关系！"
```
✅ 作业正确率：55% → +0积分（低于60%无奖励）
💪 鼓励："挫折是进步的阶梯，错题是我们的老师！"
🎯 建议：重点复习错题知识点
```

## 后置条件
1. 积分系统更新
2. 徽章成就解锁
3. 学习动力数据记录
4. 生成激励报告（可选同步家长端）

## 业务规则

### 积分规则
```
基础积分：
- 学习时长：1分钟 = 1积分（上限120分/天）
- 作业完成：基础+20，正确率>80%额外+25
- 主动提问：每次+5（鼓励主动性）
- 错题回访：正确+10，连续正确有倍数奖励

特殊奖励：
- 连续打卡：第3/7/15/30天有里程碑奖励
- 突破自我：正确率提升>20% → 额外+50
- 挑战成功：完成难度任务 → +100
```

### 徽章体系
```
等级体系：
1. 基础徽章：完成基础任务
   - 作业完成者（连续3天）
   - 提问小能手（单日提问>5次）
   
2. 进阶徽章：体现进步
   - 正确率提升王（单周提升>15%）
   - 错题终结者（回访正确率100%）
   
3. 精英徽章：长期坚持
   - 月度学习家（连续30天）
   - 学期全勤王（一学期无中断）
   
4. 特殊徽章：能力体现
   - 速度之王（速算挑战前10%）
   - 深度思考者（难题攻克）
```

### 防沉迷规则
1. **每日上限**：积分获取上限500/天
2. **时间保护**：学习超过2小时不再奖励时长积分
3. **健康提醒**：每45分钟提醒休息
4. **家长控制**：家长可设置积分兑换限制
