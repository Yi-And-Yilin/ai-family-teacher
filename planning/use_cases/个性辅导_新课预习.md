# 用例：个性辅导 - 新课预习

## 场景描述
明天数学课要学"分数除法"，小明想提前预习。AI根据教材版本和学校进度，提供个性化预习指导。

## 参与角色
- **学生**：小明
- **AI家庭教师**：系统

## 前置条件
1. 小明知道明天要学的新课内容
2. 系统已记录小明的教材版本（人教版五年级下）
3. 小明有预习意愿和时间（约20分钟）

## 基本流程

### 步骤1：启动预习模式
1. 小明打开app，点击"新课预习"按钮
2. 输入预习内容："明天学分数除法"
3. 系统确认："确认预习：分数除法（人教版五年级下第3章）"

### 步骤2：预习前测
**系统**："在开始前，我们先看看你对相关知识的掌握情况："
```
🔍 关联知识检测：
1. 分数乘法掌握度：85%（良好）
2. 倒数概念理解：90%（优秀）
3. 整数除法基础：95%（扎实）

🎯 预习基础：良好，可以开始新课预习
```

### 步骤3：核心概念讲解
**系统**："分数除法的核心规则：除以一个分数等于乘以它的倒数。"
```
📚 公式：a/b ÷ c/d = a/b × d/c

💡 理解为什么：
"2/3 ÷ 1/4 意思是：2/3里面包含多少个1/4？
相当于问：2/3是1/4的多少倍？"
```

**系统用图形解释：**
- 画图展示：2/3 = 8/12，1/4 = 3/12
- 8/12 ÷ 3/12 = 8÷3 = 8/3
- 直观看到：2/3包含约2.67个1/4

### 步骤4：关键点强调
**系统**："预习时要特别注意："
```
⚠️ 易错点预警（根据大数据）：
1. 忘记取倒数 → 直接除 → 错误
2. 符号处理 → 负数除法容易错
3. 结果未化简 → 答案不简洁
4. 带分数处理 → 先化假分数再计算

✅ 正确步骤：
1. 变除为乘
2. 取倒数
3. 计算乘法
4. 化简结果
```

### 步骤5：互动练习
**系统**："我们做几道简单题感受一下："
1. "1/2 ÷ 1/4 = ?"
   **小明**："1/2 × 4/1 = 4/2 = 2"
   **系统**："正确！1/2包含2个1/4。"

2. "3/4 ÷ 1/2 = ?"
   **小明**："3/4 × 2/1 = 6/4 = 3/2"
   **系统**："正确！3/4是1/2的1.5倍。"

3. "2/3 ÷ 1/3 = ?"
   **小明**："2/3 × 3/1 = 6/3 = 2"
   **系统**："很好！2/3包含2个1/3。"

### 步骤6：联系旧知
**系统**："联系你已经掌握的知识："
```
🔄 知识连接：
1. 分数除法 vs 分数乘法：规则相似，都是分子乘分子、分母乘分母
2. 分数除法 vs 整数除法：本质相同，都是"包含除"
3. 倒数概念：预习时重点理解"为什么用倒数"

🎯 预习目标：不是完全掌握，而是：
1. 知道核心规则
2. 理解为什么这样算
3. 能做简单题目
4. 明确听课重点
```

### 步骤7：预习总结
**系统**："20分钟预习完成！总结："
```
📋 预习成果：
✅ 掌握核心：除以分数 = 乘以倒数
✅ 理解原理：包含除的概念
✅ 简单计算：能做基础题
⏳ 待深入：复杂题目、应用题

🎒 明日听课重点：
1. 老师如何解释"为什么用倒数"
2. 例题的解题步骤
3. 易错点强调
4. 应用题解法

📝 给明天自己的建议：
"听课时有疑问及时提问，
重点关注：倒数概念的理解，
课后练习注意：结果化简。"
```

### 步骤8：生成预习笔记
**系统自动生成预习笔记，可打印或保存：**
```
📓 分数除法预习笔记
📅 预习时间：2024-01-15
⏰ 用时：20分钟

🔑 核心规则：
a/b ÷ c/d = a/b × d/c

💡 关键理解：
1. 分数除法是"包含除"
2. 倒数的作用：转换运算
3. 与分数乘法的联系

⚠️ 易错提醒：
1. 勿忘取倒数
2. 注意符号
3. 结果化简

🎯 听课目标：
- 深入理解倒数概念
- 学习规范解题步骤
- 掌握应用题解法
```

## 备选流程

### 备选A：基础薄弱
如果前测发现基础不牢：
**系统**："检测到分数乘法掌握度仅60%，建议先复习："
1. "花10分钟复习分数乘法"
2. "重点：倒数概念理解"
3. "再开始分数除法预习"

### 备选B：时间有限
**小明**："我只有10分钟预习。"

**系统**："调整为快速预习模式："
1. "核心规则讲解（3分钟）"
2. "2道例题演示（3分钟）"
3. "1道自测题（2分钟）"
4. "听课重点提醒（2分钟）"

## 后置条件
1. 完成新课预习
2. 系统记录预习数据
3. 生成预习笔记
4. 标记"已预习"，明天课后可对比掌握情况

## 业务规则
1. **难度控制**：预习内容比正式课简单30%
2. **目标合理**：预习目标为"了解"而非"掌握"
3. **联系旧知**：必须连接已有知识
4. **时间管理**：建议15-25分钟，不超过30分钟
5. **效果评估**：预习后有简单测试，评估预习效果
