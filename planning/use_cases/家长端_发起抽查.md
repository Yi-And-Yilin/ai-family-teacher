# 用例：家长端 - 发起抽查

## 场景描述
小明妈妈看到学习报告显示"分数除法需加强"，决定发起一次抽查测试，检验孩子是否真正掌握。

## 参与角色
- **家长**：小明妈妈
- **系统**：AI家庭教师
- **学生**：小明（接收测试）

## 前置条件
1. 妈妈已查看今日学习报告
2. 报告显示特定知识点需要关注
3. 妈妈想确认孩子掌握情况
4. 孩子在线或稍后会使用app

## 基本流程

### 步骤1：家长发起抽查
1. 妈妈在家长端查看学习报告
2. 在"分数除法需加强"部分点击"发起抽查"按钮
3. 系统弹出抽查设置：

**抽查设置选项：**
```
📝 抽查设置
主题：分数除法（系统根据报告推荐）
题量：○ 5题（快速） ○ 10题（标准） ○ 15题（全面）
难度：○ 基础 ○ 标准 ○ 挑战
时间：预计用时 10-20分钟

附加选项：
☑️ 限时完成（建议）
☑️ 要求拍照验证（防作弊）
☐ 立即推送（如孩子在线）
☐ 设置完成期限（如：今晚9点前）
```

4. 妈妈选择：10题、标准难度、限时完成、要求拍照验证
5. 点击"生成抽查测试"

### 步骤2：系统生成测试
**系统**："正在生成个性化抽查测试..."
```
🔍 分析孩子历史数据：
- 薄弱点：忘记取倒数（40%错误）
- 易错点：符号处理（30%错误）
- 已掌握：基础计算（90%正确）

🎯 测试设计：
1. 基础题：4题（巩固核心规则）
2. 易错题：4题（针对薄弱点）
3. 综合题：2题（检验应用能力）
```

**生成完成提示**：
"抽查测试已生成！将推送给孩子完成。您可设置奖励机制鼓励孩子认真完成。"

### 步骤3：奖励设置（可选）
**系统**："建议设置完成奖励，提高孩子积极性："
```
🎁 奖励选项：
○ 积分奖励：完成+50，正确率>80%额外+30
○ 徽章奖励：完成抽查解锁"家长认可"徽章
○ 特权奖励：兑换商店折扣券
○ 自定义奖励：______

💡 推荐：积分奖励+徽章，既激励又认可
```

妈妈选择：积分奖励+徽章奖励

### 步骤4：测试推送
**系统执行推送**：
1. 如果小明在线：立即推送测试通知
2. 如果小明离线：下次打开app时推送
3. 推送消息："妈妈发来了抽查测试，完成后有奖励哦！"

**同时给妈妈确认**：
"抽查测试已发送！孩子完成后您将收到报告。预计完成时间：20分钟内。"

### 步骤5：孩子端接收测试
**小明收到通知**：
```
🔔 新消息：妈妈发来抽查测试
📝 内容：分数除法抽查（10题）
⏰ 建议用时：20分钟
🎁 完成奖励：积分+50，"家长认可"徽章
📸 要求：需要拍照验证

[开始测试] [稍后做]
```

小明点击"开始测试"，进入测试界面。

### 步骤6：测试过程（孩子端）
**测试界面**：
```
📋 家长抽查测试
👩 发起人：妈妈
🎯 主题：分数除法
📊 进度：0/10题
⏱️ 用时：0:00（限时25分钟）

📸 拍照要求：
- 每道题需在纸上计算
- 完成一题拍一题
- 系统验证计算过程
```

**题目示例**：
1. "基础题：3/4 ÷ 1/2 = ?（拍照计算过程）"
2. "易错题：2/3 ÷ (-1/4) = ?（注意符号）"
3. "综合题：小明有3/4升果汁，每杯1/6升，可分几杯？列式计算"

小明逐一做题、拍照上传。

### 步骤7：系统批改与验证
**系统实时处理**：
1. **答案验证**：检查计算结果
2. **过程验证**：通过拍照验证：
   - 是否在纸上计算
   - 计算步骤是否合理
   - 是否有作弊嫌疑（如直接抄答案）
3. **即时反馈**：每做完一题，显示"✓ 正确"或"✗ 错误，正确答案：..."

### 步骤8：测试完成与报告
**小明完成所有题目后**：
```
🎉 抽查测试完成！
📊 成绩：8/10题正确（80%）
⏰ 用时：18分钟（在规定时间内）
📸 验证：拍照验证通过

🏆 获得奖励：
- 积分：+50（完成）+20（正确率>80%）= +70
- 徽章："家长认可"（已解锁）
```

**系统生成详细报告**，准备发送给家长。

### 步骤9：家长接收报告
**妈妈收到推送**：
"小明已完成抽查测试，成绩：80%，用时18分钟。查看详细报告"

**详细报告内容**：
```
📋 抽查测试报告
👦 测试人：小明
👩 发起人：妈妈
📅 完成时间：2024-01-15 20:45

📊 总体结果：
- 题量：10题
- 正确：8题（80%）
- 用时：18分钟（效率良好）
- 验证：拍照验证通过（无作弊嫌疑）

🔍 知识点分析：
✅ 掌握良好：基础计算（4/4正确）
⚠️ 仍需加强：符号处理（2/4正确）
❌ 薄弱点：应用题理解（1/2正确）

📝 具体表现：
1. 基础题全对 → 核心规则已掌握
2. 符号题50%正确 → 负数运算需巩固
3. 应用题50%正确 → 实际应用能力待提高

💡 给家长的建议：
1. 肯定：基础扎实，态度认真（拍照验证配合）
2. 建议：重点练习负数分数除法
3. 行动：可安排2-3道应用题周末练习
4. 放心：整体掌握度良好，无需过度担心
```

### 步骤10：家长后续操作
妈妈查看报告后可以：
1. **发送鼓励**：一键发送鼓励消息
2. **安排练习**：根据薄弱点生成专项练习
3. **沟通老师**：如有疑问，可留言给AI老师
4. **调整监督**：根据测试结果调整后续监督策略

## 备选流程

### 备选A：孩子拒绝或拖延
如果小明点击"稍后做"或一直未完成：
**系统给妈妈提醒**：
"提醒：抽查测试已发送2小时，孩子尚未开始。"
**选项**：
1. 发送提醒消息
2. 取消本次抽查
3. 调整截止时间

### 备选B：拍照验证不通过
如果系统检测到拍照有问题：
**系统给小明提示**：
"拍照验证未通过，请确保："
1. "在纸上独立计算"
2. "清晰拍摄计算过程"
3. "不要直接拍答案"

**给妈妈的特殊报告**：
"⚠️ 拍照验证异常：可能未独立完成。建议关注。"

## 后置条件
1. 完成抽查测试闭环
2. 家长获得真实掌握情况
3. 孩子获得相应奖励
4. 系统更新学习数据

## 业务规则
1. **测试设计**：基于历史数据的个性化出题
2. **防作弊机制**：拍照验证+过程验证
3. **时间管理**：合理限时，避免压力过大
4. **奖励平衡**：奖励适度，避免功利化
5. **隐私保护**：测试内容仅家长和孩子可见
