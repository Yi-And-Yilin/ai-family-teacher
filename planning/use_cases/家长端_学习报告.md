# 用例：家长端 - 学习报告查看

## 场景描述
晚上9点，小明妈妈结束工作，想了解孩子今天的学习情况。她打开家长端app，查看AI生成的每日学习报告。

## 参与角色
- **家长**：小明妈妈
- **系统**：AI家庭教师家长端

## 前置条件
1. 小明已完成今天的学习（作业+复习）
2. 系统已生成当日学习报告
3. 妈妈已安装家长端app并绑定孩子账号

## 基本流程

### 步骤1：打开家长端
1. 妈妈打开"AI家庭教师家长端"app
2. 看到主界面概览：
   ```
   👦 小明今日学习状态
   ✅ 作业完成：数学作业20题
   ⏰ 学习时长：45分钟
   📊 掌握情况：良好
   🔔 新消息：今日学习报告已生成
   ```

### 步骤2：查看每日报告
妈妈点击"今日报告"卡片，进入详细报告页：

**报告标题**：小明今日学习报告（2024-01-15）

**第一部分：学习概览**
```
📅 报告时间：2024-01-15 20:30
⏰ 总学习时长：45分钟
📚 学习内容：数学作业（分数运算）
🎯 完成情况：作业20题完成，复习15题完成

📊 核心数据：
- 作业正确率：85%（17/20）
- 复习正确率：80%（12/15）
- 提问次数：3次（遇到难题主动求助）
- 专注度评分：8.5/10
```

**第二部分：知识点分析**
```
🔍 知识点掌握情况：
✅ 已掌握：分数加减法（通分熟练）
⚠️ 需关注：分数除法（偶尔忘记取倒数）
📈 进步点：带分数转换错误减少50%

📝 具体表现：
1. 分数加法：10题全对，掌握牢固
2. 分数减法：8题对7题，小失误
3. 分数乘法：5题对4题，良好
4. 分数除法：7题对5题，需加强
```

**第三部分：错题分析**
```
❌ 今日错题（3题）：
1. 2/3 ÷ 1/4 = 8/3（未写带分数格式）
   💡 AI已讲解，小明理解
   
2. 3/4 - 1/6 = 7/12（通分计算错误）
   💡 粗心错误，已提醒检查
   
3. 应用题：分果汁问题列式错误
   💡 理解题意有困难，建议多练应用题
```

**第四部分：AI观察与建议**
```
👀 AI老师观察：
"小明今天学习状态很好，遇到难题主动提问3次。
在分数除法上表现出犹豫，需要巩固概念。
应用题理解需要加强，建议从生活实例入手。"

🎯 给家长的建议：
1. 肯定：表扬孩子主动求助的态度
2. 提醒：提醒检查计算步骤，避免粗心错误
3. 建议：周末可做2-3道分数除法应用题巩固
4. 放心：整体掌握良好，无需过度担心
```

**第五部分：可视化图表**
- 正确率趋势图（最近7天）
- 知识点掌握度雷达图
- 学习时长分布图
- 错题类型饼图

### 步骤3：查看详细记录
妈妈想了解具体某道题的讲解过程，点击错题1的"查看详情"：

**错题详情页**：
```
题目：2/3 ÷ 1/4 = ?
小明答案：8/3
正确答案：2又2/3（或8/3）

AI讲解记录：
17:15 小明提问
17:16 AI：答案正确，但建议写带分数格式
17:17 AI：8/3 = 6/3 + 2/3 = 2 + 2/3
17:18 AI：练习：12/5 = ? 
17:19 小明：2又2/5 ✓
17:20 AI：掌握，标记为"格式需注意"

掌握状态：已理解，需注意格式规范
```

### 步骤4：发起家长操作
妈妈看到报告后，可以：
1. **一键鼓励**：点击"发送鼓励"按钮，系统自动发送鼓励消息给孩子
2. **设置提醒**：针对"分数除法需加强"，设置"明天提醒孩子复习"
3. **发起抽查**：点击"出题测试"，系统生成5道分数除法题，明天让孩子做
4. **联系老师**：如有疑问，可留言给AI老师

### 步骤5：周报预览
妈妈滑动到页面底部，看到周报预览：

```
📅 本周学习趋势（1.8-1.15）：
📈 整体进步：分数运算掌握度从70%提升到85%
🏆 亮点：连续7天完成作业
🎯 重点关注：应用题理解能力

📋 建议周末安排：
1. 复习分数除法概念（30分钟）
2. 做3道生活应用题（20分钟）
3. 错题回访：带分数转换（10分钟）
```

## 备选流程

### 备选A：学习状态异常
如果系统检测到异常：
```
⚠️ 今日关注：学习时长仅15分钟
🔍 可能原因：作业较少 or 中途放弃
💡 建议：询问孩子是否遇到困难
```

### 备选B：需要家长介入
如果发现严重问题：
```
🚨 需要关注：同一知识点连续3天出错
📝 问题：分数除法概念混淆
👨‍👦 建议行动：今晚与孩子一起看AI讲解记录
📞 可联系：AI老师已准备专项练习
```

## 后置条件
1. 家长了解孩子学习情况
2. 可采取相应行动（鼓励、提醒、测试）
3. 系统记录家长查看行为
4. 如有操作，同步到孩子端

## 业务规则
1. **报告时间**：每日20:30自动生成
2. **内容分级**：
   - 绿色：正常，无需特别关注
   - 黄色：需留意，建议提醒
   - 红色：需介入，建议行动
3. **隐私保护**：不显示具体题目内容（除非家长点击详情）
4. **正向引导**：以鼓励和建议为主，不制造焦虑

## 界面示意
```
[家长端首页]
👦 孩子1：小明
   ✅ 今日作业完成
   📊 查看详细报告
   
👧 孩子2：小红（如有多个）
   ⏳ 学习中...

[报告页面]
📅 2024-01-15 学习报告
📊 数据卡片（正确率、时长、专注度）
📈 趋势图表
📝 错题摘要
💡 AI建议

[操作栏]
👍 发送鼓励
🔔 设置提醒
📝 发起抽查
💬 联系AI老师
```

## 数据安全
1. **数据脱敏**：报告中的具体题目需要二次确认才显示
2. **访问控制**：只有绑定家长可查看
3. **历史记录**：报告保存30天，可随时回顾
4. **导出功能**：支持导出PDF，用于家长会等场合
